# Hedgehog RP TV
> **Основной проект:** [hedgehog-rp](https://github.com/MaSStiK/hedgehog-rp)

**Ежиное телевидение** - видеоплатформа вселенной Hedgehog RP, созданная для хранения и просмотра всех выпусков «Ежиных Хроник». Сервис объединяет видеоматериалы разных периодов существования проекта и позволяет сохранять историю, ключевые события и атмосферу ролевой вселенной в формате единого видеоархива.

Платформа использовалась как центральное хранилище видеоконтента сообщества и обеспечивала удобный доступ к архиву выпусков для всех участников проекта.

## ✨ Возможности
- Просмотр всех выпусков «Ежиных Хроник»
- Удобная навигация по видеоархиву
- Доступ к материалам разных сезонов и периодов проекта
- Централизованное хранение видеоконтента сообщества
- Отображение информации о выпусках и датах публикации
- Адаптивный интерфейс для различных устройств

## 🛠️ Технологии
- **Next.js** — основа приложения и роутинг
- **React** — интерфейс приложения
- **clsx** — объединение CSS-классов
- **Tippy.js** — всплывающие подсказки и интерактивные элементы
- **Moment.js** — работа с датой и временем

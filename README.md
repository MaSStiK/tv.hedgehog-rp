# Hedgehog RP TV
> **Основной проект:** [hedgehog-rp](https://github.com/MaSStiK/hedgehog-rp)

**Ежиное телевидение** - видеохостинг вселенной Hedgehog RP, разработанный для хранения и просмотра всех серий «Ежиных Хроник».  
Сайт позволяет удобно просматривать выпуски, собранные за разные периоды существования проекта, сохраняя историю, события и атмосферу ролевой вселенной Hedgehog RP в формате видеоархива.

Платформа использовалась как центральное хранилище видеоконтента сообщества.

## 🔗 Связанные проекты
- 🌍 [Hedgehog RP](https://github.com/MaSStiK/hedgehog-rp)
- 🌍 [Онлайн карта](https://github.com/MaSStiK/map.hedgehog-rp)
- 📊 [Статистика беседы](https://github.com/MaSStiK/stats.hedgehog-rp)

## 🛠️ Технологии
- **Next.js** — основа приложения и роутинг
- **React** — интерфейс приложения
- **@tippyjs/react** — всплывающие подсказки и интерактивные элементы
- **clsx** — удобное объединение CSS-классов
- **moment** — работа с датой и временем
